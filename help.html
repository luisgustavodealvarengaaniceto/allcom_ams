Visão Geral do Sistema

Consulta de Status

Consulte o status de equipamentos Allcom JC450 via IMEI, com informações detalhadas de firmware, conectividade e última comunicação.

Dashboard Analítico

Visualize métricas em tempo real, gráficos de distribuição e análise de status dos equipamentos com relatórios detalhados.

Gestão de Firmware

Configure firmware de referência, visualize distribuição por versão e identifique equipamentos que precisam de atualização.

Exportação Avançada

Exporte dados em formato CSV, Excel ou JSON com formatação profissional e campos organizados para análise.

Especificações do Sistema

Processamento: Automático em lotes de 99 IMEIs
Formatos Suportados: .txt, .csv para importação
Exportação: CSV, Excel (.xlsx), JSON
API: JimiCloud FOTA API
Timezone: Conversão automática China → Brasil

Guia de Uso Passo a Passo

1

Configurar Firmware de Referência

Antes de consultar, configure o firmware de referência no dropdown. Isso permite comparar as versões dos equipamentos.

  • Selecione uma versão existente
  • Ou adicione uma nova versão clicando em "Adicionar novo firmware"
  • Formato: C450pro_2.0.16_20250526
2

Inserir IMEIs

Digite os IMEIs na caixa de texto (um por linha) ou carregue um arquivo .txt/.csv.

Exemplo: 869247060051882
868120302622664
868120302622665
Dica: Você pode inserir quantos IMEIs precisar. O sistema processa automaticamente em lotes.
3

Validação Automática

O sistema valida automaticamente cada IMEI (15 dígitos numéricos). IMEIs inválidos são destacados e não processados.

4

Iniciar Consulta

Clique em "Consultar Status" para iniciar o processamento. O sistema divide automaticamente em lotes de 99 IMEIs.

5

Acompanhar Progresso

Observe a barra de progresso detalhada que mostra:

  • Status atual do processamento
  • Porcentagem de conclusão
  • Número de lotes processados
6

Analisar Resultados

Visualize os resultados na lista de equipamentos com:

  • Status de conectividade (Online/Offline)
  • Versão do firmware
  • Última comunicação
  • Dias offline (se aplicável)
7

Usar Filtros e Busca

Utilize os filtros para refinar a visualização:

  • Filtro por status: Todos/Online/Offline
  • Busca por IMEI específico
  • Clique em qualquer equipamento para detalhes completos

Gestão de Firmware

Configuração de Firmware de Referência

O sistema permite configurar um firmware de referência para comparação automática com os equipamentos consultados.

Adicionar Novo Firmware

Selecione "Adicionar novo firmware" no dropdown para inserir uma nova versão de referência.

Formato esperado: C450pro_2.0.16_20250526

Versões Disponíveis

O sistema mantém um histórico de versões utilizadas:

  • C450Pro_2.0.16_20250526 (Mais Recente)
  • C450Pro_2.0.13_20250410
  • C450Pro_2.0.08_20241028

Detecção Automática

O sistema extrai automaticamente a versão do firmware dos equipamentos consultados, buscando por:

  • Campo "version" na resposta da API
  • Parâmetro "C450" no campo "selfCheckParam"
  • Busca robusta por qualquer campo que comece com "C450"

Dashboard de Firmware

No dashboard, você pode visualizar a distribuição de firmware dos equipamentos consultados.

Cards Clicáveis

Cada card de firmware no dashboard é clicável. Ao clicar, você verá:

  • Lista detalhada de todos os IMEIs com aquela versão
  • Status de conectividade de cada equipamento
  • Última comunicação e dias offline
  • Opção para exportar apenas os equipamentos dessa versão

Exportação por Versão

No modal de detalhes do firmware, você pode exportar apenas os equipamentos daquela versão específica em formato CSV.

Dashboard Analítico

Visualização e Controle

O dashboard fornece uma visão analítica completa dos equipamentos consultados, com métricas em tempo real e gráficos interativos.

Métricas Gerais

  • Total de Equipamentos: Quantidade total consultada
  • Equipamentos Online: Dispositivos com comunicação recente
  • Equipamentos Offline: Dispositivos sem comunicação
  • Média de Dias Offline: Tempo médio sem comunicação
  • Problemas de Armazenamento: Equipamentos com cartão SD acima de 90%

Status de Conectividade

Gráfico visual mostrando a distribuição entre equipamentos online e offline, com cores distintivas.

Distribuição de Firmware

Cards clicáveis mostrando cada versão de firmware encontrada:

  • Versão do firmware
  • Quantidade de equipamentos
  • Indicador se está atualizado (comparado com firmware de referência)
  • Clique para ver detalhes dos equipamentos

Status de Armazenamento

Análise do uso de cartão SD dos equipamentos:

  • Equipamentos com armazenamento normal
  • Equipamentos com uso alto (acima de 90%)
  • Distribuição por tipo de armazenamento (SD1, SD2, Memória)

Controles do Dashboard

Mostrar/Ocultar

Use o botão "Mostrar Dashboard" para exibir ou ocultar a visualização analítica.

Atualização Automática

O dashboard se atualiza automaticamente a cada nova consulta realizada.

Exportação de Dados

Formatos Disponíveis

O sistema oferece três formatos de exportação para atender diferentes necessidades de análise:

CSV (Comma-Separated Values)

Formato ideal para análise em planilhas e importação em outros sistemas.

  • Compatível com Excel, Google Sheets
  • Separador: ponto e vírgula (;)
  • Codificação: UTF-8 com BOM
  • Formatação brasileira de datas

Excel (.xlsx)

Formato nativo do Microsoft Excel com formatação avançada.

  • Formatação automática de colunas
  • Cores para status (verde=online, vermelho=offline)
  • Largura otimizada das colunas
  • Compatível com Excel 2007+

JSON (JavaScript Object Notation)

Formato para integração com sistemas e análise programática.

  • Dados estruturados completos
  • Inclui metadados da consulta
  • Timestamp da exportação
  • Ideal para APIs e sistemas

Campos Exportados

Todos os formatos incluem os seguintes campos:

IMEI: Identificador único do equipamento
Status: Online ou Offline
Versão do Firmware: Versão atual do equipamento
Última Comunicação: Data/hora da última comunicação
Dias Offline: Quantidade de dias sem comunicação
Cartão SD: Status do armazenamento
Dados Técnicos: Informações completas do equipamento

Tipos de Exportação

Exportação Completa

Exporta todos os equipamentos consultados com todos os dados disponíveis.

Exportação por Firmware

No modal de detalhes do firmware, exporta apenas os equipamentos de uma versão específica.

Solução de Problemas

Problemas Comuns e Soluções

Problemas de Consulta

Consulta não funciona ou trava

Soluções:

  • Verifique se os IMEIs estão corretos (15 dígitos)
  • Reduza a quantidade de IMEIs por consulta
  • Aguarde alguns segundos e tente novamente
  • Verifique conexão com a internet
Resultados inconsistentes

Soluções:

  • Aguarde e refaça a consulta
  • Verifique se a API está funcionando
  • Limpe cache do sistema
  • Consulte IMEIs em lotes menores

Problemas de Dashboard

Dashboard não carrega ou mostra dados incorretos

Soluções:

  • Recarregue a página
  • Limpe cache do navegador
  • Refaça a consulta dos IMEIs
  • Verifique se há dados suficientes
Cards de firmware não são clicáveis

Soluções:

  • Aguarde o carregamento completo
  • Recarregue a página
  • Verifique se JavaScript está habilitado
  • Tente um navegador diferente

Problemas de Exportação

Exportação não funciona ou arquivo corrompido

Soluções:

  • Verifique se há dados para exportar
  • Tente um formato diferente (CSV, Excel, JSON)
  • Aguarde alguns segundos e tente novamente
  • Verifique permissões de download do navegador
Arquivo exportado não abre corretamente

Soluções:

  • Para CSV: Abra com Excel usando importação de dados
  • Para Excel: Verifique se tem Excel 2007+ instalado
  • Para JSON: Use editor de texto ou ferramenta JSON
  • Verifique se o download foi concluído

Suporte Técnico

Se os problemas persistirem:

  • Anote o erro específico (se houver)
  • Documente os passos que levaram ao problema
  • Verifique o console do navegador (F12)
  • Entre em contato com o suporte técnico
